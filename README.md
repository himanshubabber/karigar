# karigar
link : https://karigar-mu.vercel.app/

Karigar is a full-stack web application that connects customers with skilled local workers like plumbers, electricians, and carpenters. Customers can create service requests, track workers in real time on a map, and manage payments and ratings. Workers receive job requests instantly, share their live location, and manage their profiles and earnings. The platform makes hiring local services as seamless as booking a cab.
